✨ NX Monorepo / Multi App ✨

Mi profile es un monorepo Angular 21 con modelado reactivo

Son librerías no buildeables que las utilizo para explorar y validar nuevas features de Angular (Signals, Signal-based Forms, Resources, RxResource, etc.).

Es un constante modelo en crecimiento por lo que encontrás @todos, integraciones con fake APIs u otros procesos en construcción o experimentales.

⭐ App Profile

Quick Start

nx serve app/profile --open

⭐ Libraries

  • Infrastructure AuthInterceptor, UserDomainModel, AuthUser, UsersResponse

⭐ Library Types

  • domain: Core business logic and domain models
  • infrastructure: HTTP interceptors, guards, services, API clients
  • features: Feature-specific functionality
  • state: NgRx (actions, reducers, effects, selectors)
  • ui: Reusable presentational components
  • utils: Shared helpers, pipes
  • shared: Cross-cutting libraries
