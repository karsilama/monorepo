# API

A NestJS-based API application built with Webpack, part of the monorepo.

## Getting Started

Start the development server:

```bash
nx serve api
```

The API will be running and ready to handle requests.

## Overview

- Built with **NestJS** for robust API development
- Uses **Webpack** for bundling
- Includes HTTP module for external integrations
- Follows the standard NestJS module architecture (controllers, services, modules)
